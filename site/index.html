dot-kanban

Kanban-driven AI coding for Claude Code

Quick start npx dot-kanban

What it does

  • Structured workflow — gives Claude a kanban board to plan and track its own work, reducing drift and wasted tokens
  • Phase tracking — enforces an analyze → locate → implement → verify loop so nothing gets skipped
  • Minimal, focused changes — each task is scoped tightly, keeping diffs small and reviewable
  • Zero config — installs as a Claude Code plugin, no setup required

Install

npx

npx dot-kanban

Claude Code plugin

/install-plugin dylangroos/claude-kanban
